Plan Details

Free Plan

$0/mo — No credit card required
  • Code completions2,000/mo
  • Chat messages50/mo
  • Agent modeNo
  • Available in VS Code and JetBrains
  • Basic code suggestions and inline completions
  • Good enough to evaluate if Copilot fits your workflow
  • No access to premium models or agent features

Pro Plan

$10/mo ($100/yr) — Best value for solo developers
  • Code completionsUnlimited
  • Chat messagesUnlimited
  • Agent modeLimited
  • Unlimited completions and chat — the rate limits are gone
  • Multiple model access (GPT-4o, Claude 3.5 Sonnet)
  • Works across VS Code, JetBrains, Neovim, Xcode, Visual Studio
  • At $10/mo, cheapest paid AI coding tool on the market

Pro+ Plan

$39/mo — For developers who use agent mode heavily
  • Code completionsUnlimited
  • Chat messagesUnlimited
  • Agent modeUnlimited
  • Access to premium models: Claude Opus, o1-preview
  • Unlimited Copilot agent mode (Copilot Workspace)
  • Best for developers doing complex, multi-file tasks
  • Introduced 2025 to compete with Cursor's agent features

Business Plan

$19/seat/mo — Team management and policies
  • Per-seat price$19/mo
  • Min seatsNone
  • Annual billing$228/yr
  • Organization-wide license management
  • IP indemnity and code training opt-out
  • Policy management (block certain suggestions)
  • Usage metrics and audit logging

Enterprise Plan

$39/seat/mo — Knowledge bases and fine-tuning
  • Per-seat price$39/mo
  • Annual billing$468/yr
  • SSO/SAMLIncluded
  • Everything in Business plus fine-tuning on your codebase
  • Knowledge bases for internal documentation
  • SAML SSO and advanced security controls
  • Dedicated support and SLA
