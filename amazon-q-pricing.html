Amazon Q Developer Pricing: Both Plans Explained

Amazon Q Developer keeps it simple: two tiers, no credits, no usage caps to worry about. The Free tier is generous enough for individual developers. Pro adds the admin controls and higher limits that teams need.

PlanPriceAnnualKey Features
Free (Individual) $0 $0 Code suggestions, chat, security scans (50/mo), CLI access
Pro $19/mo $228/yr ($19/mo) Higher limits, admin controls, organizational policies, custom transformations

No annual discount: Unlike most competitors, Amazon Q Pro is $19/mo whether you pay monthly or annually. The upside: no commitment lock-in, cancel anytime.

What's Included in Free

Amazon Q's free tier is one of the most capable among AI coding tools. You get real features, not a trial.

Code Completions

Inline suggestions across 15+ languages
  • Python, JavaScript, TypeScript, Java, C#, Go, Rust, and more
  • Single-line and multi-line completions
  • Context-aware suggestions from your open files
  • Works in VS Code and JetBrains IDEs

Amazon Q Chat

AI assistant in your IDE
  • Ask questions about your code, AWS services, or general programming
  • Explain code, generate code, and suggest refactors
  • AWS-specific knowledge built in (IAM policies, CloudFormation, etc.)
  • Conversational context within your workspace

Security Scanning

50 scans per month — included free
  • Scans for vulnerabilities, secrets, and misconfigurations
  • Covers OWASP Top 10 and CWE categories
  • Actionable remediation suggestions
  • Supports Python, Java, JavaScript, TypeScript, C#, and more

CLI Access

Amazon Q in the terminal
  • Natural language to CLI command translation
  • Autocomplete for AWS CLI, git, and common tools
  • Context-aware suggestions based on your shell history
  • macOS and Linux support

What Pro Adds

Pro is for teams that need admin oversight, higher limits, and access to Amazon Q's most powerful features: code transformation and agent capabilities.

Higher Usage Limits

More of everything
  • Increased code completion and chat request limits
  • Unlimited security scans (vs. 50/mo on Free)
  • Priority access during peak usage

Admin Dashboard

Centralized team management
  • AWS IAM Identity Center integration for SSO
  • Usage analytics and reporting per developer
  • Centralized billing through your AWS account

Organizational Controls

Policies and access management
  • Define which features developers can access
  • Set organizational policies for code suggestions
  • Control data sharing and telemetry settings
  • Manage reference tracking for open-source code

Code Transformation

Automated modernization at scale
  • Java 8 to Java 17 automated upgrades
  • .NET Framework to cross-platform .NET Core migration
  • Handles dependency updates, API changes, and test generation
  • Agent-driven: reviews, plans, and executes multi-file changes

Best For

Amazon Q Developer is not for everyone. Here's where it genuinely shines.

AWS-Heavy Teams

  • Native understanding of AWS services, APIs, and best practices
  • Generate CloudFormation, CDK, and Terraform for AWS resources
  • Billing consolidated into your existing AWS account

Java Modernization

  • Automated Java 8 to Java 17 upgrades — not just syntax, full dependency migration
  • .NET Framework to .NET Core transformation
  • Saves weeks of manual migration work per application

Security-Conscious Organizations

  • Built-in security scanning included even on the free tier
  • Vulnerability detection aligned with OWASP and CWE standards
  • No data leaves AWS — stays within your security perimeter

Teams Already on AWS

  • IAM Identity Center integration for SSO — no separate accounts needed
  • Organizational policies managed through AWS console
  • Usage billed as an AWS service — one vendor, one invoice
