GitHub Copilot Pricing: Every Plan Explained

GitHub Copilot now has 5 tiers — from a free plan to a $39/seat enterprise option. The key mechanic: "premium requests" that meter access to advanced AI models.

PlanPriceCompletionsPremium RequestsCoding AgentBest For
Free $0 2,000/mo 50/mo No Students, hobbyists
Pro $10/mo Unlimited 300/mo Yes Solo developers
Pro+ $39/mo Unlimited 1,500/mo Yes Power users, agent-heavy workflows
Business $19/seat/mo Unlimited 300/user/mo Yes Teams, organizations
Enterprise $39/seat/mo Unlimited 1,000/user/mo Yes Large orgs, knowledge bases

Included models (free, unlimited on paid plans): GPT-4.1, GPT-4o, GPT-5 mini. Premium requests are only consumed by advanced models like Claude Sonnet, Opus, and Gemini Pro. Overage: $0.04/request on all paid tiers.

Premium Request Multipliers — The Hidden Cost

Not all models cost the same. Each model has a multiplier that determines how many premium requests it consumes per interaction. This is the most important thing to understand about Copilot pricing.

MultiplierModelsPro (300 req) = interactionsPro+ (1,500 req) = interactions
0x (free) GPT-4.1, GPT-4o, GPT-5 mini Unlimited Unlimited
0.25x Grok Code Fast 1 1,200 6,000
0.33x Claude Haiku 4.5, Gemini 3 Flash, GPT-5.4 mini ~909 ~4,545
1x Claude Sonnet 4/4.5/4.6, Gemini 2.5/3/3.1 Pro, GPT-5.1/5.2/5.4 300 1,500
3x Claude Opus 4.5, Claude Opus 4.6 100 500
30x Claude Opus 4.6 (fast mode preview) 10 50

Tip: Let Copilot auto-select the model to get a 10% discount (0.9x multiplier). The "Auto" mode routes to the cheapest effective model.

Plan Details

Free Plan

$0/mo — No credit card required
  • Code completions2,000/mo
  • Premium requests50/mo
  • Coding agentNo
  • Available in VS Code, JetBrains, and GitHub.com
  • Included models: GPT-4.1, GPT-4o, GPT-5 mini, Claude Haiku 4.5
  • Good enough to evaluate if Copilot fits your workflow
  • Also free for verified students, teachers, and OSS maintainers

Pro Plan

$10/mo ($100/yr) — Best value for solo developers
  • Code completionsUnlimited
  • Premium requests300/mo
  • Coding agentYes
  • Unlimited completions + unlimited chat with included models
  • 300 premium requests = 300 Sonnet or 100 Opus interactions
  • Works across VS Code, JetBrains, Neovim, Xcode, Visual Studio
  • Overage: $0.04/request. At $10/mo, cheapest paid AI coding tool

Pro+ Plan

$39/mo ($390/yr) — 5x the premium requests of Pro
  • Code completionsUnlimited
  • Premium requests1,500/mo
  • GitHub SparkYes
  • All models including Claude Opus 4.6 fast mode (preview)
  • 1,500 requests = 500 Opus interactions or 50 Opus-fast
  • GitHub Spark access (AI app builder)
  • Worth it if you use 1,000+ premium requests/mo on Pro

Business Plan
